Welcome to Anshal's chatapp simplified version , this chat app is built using react and nodejs as backend with docker 

How to Get Started- 
in the client Dir run-
    npm install 

in the server dir run
    npm install 
    create a .env file and include PORT number
    start the server by-
    npm run dev for development server or
    npm start for production


in the client dir run-
    npm start to start the app


Happy Hacking
